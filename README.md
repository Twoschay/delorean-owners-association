# DeLorean Owners Association (DOA) — Demo Website

> Built by Tuschay Studios as a full-featured demo website.

## Overview

A modern Next.js 14 replacement for the DOA website — featuring a complete member experience with mock data, interactive features, and a beautiful "Stainless Luxe" design system inspired by the DMC-12's iconic stainless steel body.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS v3 + shadcn/ui
- **Design System**: "Stainless Luxe" — dark theme with amber accents
- **Mock Auth**: Context-based demo auth system
- **Mock Data**: 250 members, 50 vehicles, 20 magazine issues, 12 events

## Design System

**Color Palette (Stainless Luxe):**
- Steel: `#C0C0C0` — brushed stainless primary
- Amber: `#F5A623` — turn signal accent color
- Obsidian: `#1A1A1A` — deep background
- Charcoal: `#2D2D2D` — card surfaces

**Typography:**
- Display: Bebas Neue (headers)
- Body: DM Sans
- Mono: JetBrains Mono (VINs, specs)
